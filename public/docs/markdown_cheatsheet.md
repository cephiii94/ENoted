# ENoted Markdown Cheatsheet

Selamat datang di Panduan Penulisan ENoted! Gunakan panduan ini untuk mempercantik artikel Anda.

## 1. Judul (Headers)
Gunakan tanda `#` diikuti spasi untuk membuat judul.
```markdown
# Judul Utama (H1)
## Sub Judul (H2)
### Judul Bagian (H3)
```

## 2. Penekanan (Emphasis)
```markdown
**Tebal** atau __Tebal__
*Miring* atau _Miring_
***Tebal & Miring***
~~Coretan (Strikethrough)~~
```

## 3. Daftar (Lists)
### Daftar Tidak Berurutan:
```markdown
- Item satu
- Item dua
  - Sub-item
```
### Daftar Berurutan:
```markdown
1. Langkah pertama
2. Langkah kedua
```

## 4. Tautan & Gambar
```markdown
[Teks Tautan](https://www.enoted.com)
![Deskripsi Gambar](https://link-gambar.jpg)
```

## 5. Kutipan (Blockquotes)
```markdown
> "Berbagi ilmu adalah cara terbaik untuk belajar." - ENoted
```

## 6. Kode (Code)
### Kode Inline:
Gunakan backtick ( \` ) untuk `kode dalam kalimat`.

### Blok Kode:
Gunakan tiga backtick ( \` \` \` ) untuk blok kode besar:
```javascript
function helloWorld() {
  console.log("Halo ENoted!");
}
```

## 7. Tabel (Tables)

